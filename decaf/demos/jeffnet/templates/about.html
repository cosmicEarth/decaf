About the Demo

This demo is created by Yangqing Jia and Jeff Donahue at University of California, Berkeley. The demo code lives purely in the realm of Python, with flask as the front-end server.

The classifier is a convolutional neural network trained using data from ImageNet with 1,000 classes. You can view the training data here. The images are reshaped so that the shorter side has length 256, and the centre 256x256 part is cropped for training. For each image, we are showing the top 5 predictions as well as their scores among the 1000 classes.

About Decaf

Decaf is a general python framework for deep convolutional neural networks. It relies on Python and a set of numerical computation modules (such as numpy/scipy) to perform efficient, CPU-based computation as well as distributed optimization of various networks.
